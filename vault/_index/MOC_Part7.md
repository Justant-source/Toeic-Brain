## Section 1: 독해 전략 (p.336–393)

> 문제 유형 파악 → 질문 먼저 읽기 → 스캐닝·스키밍 활용

- [ ] Chapter 01: 주제·목적 파악 문제 (What is the purpose of…?)
- [ ] Chapter 02: 세부 사항 파악 문제 (According to the …)
- [ ] Chapter 03: NOT/TRUE 문제 (Which is NOT mentioned…?)
- [ ] Chapter 04: 추론 문제 (What is implied/suggested…?)
- [ ] Chapter 05: 동의어 문제 (The word "X" in … is closest in meaning to…)
- [ ] Chapter 06: 문장 삽입 위치 문제 ([1] / [2] / [3] / [4])
- [ ] Chapter 07: 화자 의도 파악 문제 (What does … mean by saying…?)

### 핵심 전략 요약

| 문제 유형 | 전략 |
|-----------|------|
| 주제·목적 | 첫 단락 집중, 이메일 제목(Subject) 확인 |
| 세부 사항 | 키워드 스캐닝 → 해당 위치 정독 |
| NOT/TRUE | 4개 선택지 모두 지문과 대조 (시간 소요 주의) |
| 추론 | 직접 기술되지 않음 — 문맥과 함의 파악 |
| 동의어 | 문장 속 문맥 의미 기준 (사전적 의미 오답 주의) |
| 문장 삽입 | 연결어(therefore, however, in addition)·대명사 흐름 확인 |
| 화자 의도 | 전후 맥락 파악 — 표면적 의미 ≠ 의도 |

### 복수 지문 핵심 전략

- **연계 문제**: 두 지문을 교차 참조해야 풀리는 문제 (정보를 한 지문에서만 찾으면 오답)
- **지문 읽기 순서**: 질문 확인 → 첫 번째 지문 → 두 번째(세 번째) 지문
- **시간 관리**: 복수 지문에 45~50초/문항 배분 권장
