# 가목적어 it 구문

> [!info] 출처
> 해커스 토익 RC 기본서 (2023) — Section 1 문장패턴 > Chapter 2 목적어·보어·수식어 > 항목 2 (p.56)

## 핵심 개념

5형식 동사(make, find, consider, think 등)에서 진목적어가 to부정사구나 that절일 때, **it**을 가목적어로 쓰고 진목적어를 뒤로 보낸다.

`동사 + it + 형용사/명사(목적격보어) + to부정사/that절(진목적어)`

## 핵심 공식/패턴

```
S + make/find/consider + it + 형용사 + to부정사/that절
```

## 예문

| # | 예문 | 해석 | 포인트 |
|---|------|------|--------|
| 1 | The manager made **it** clear **that** all reports must be submitted. | 관리자는 모든 보고서를 제출해야 한다는 것을 명확히 했다. | it = 가목적어, that절 = 진목적어 |
| 2 | We found **it** difficult **to** complete the project on time. | 우리는 제시간에 프로젝트를 완료하는 것이 어렵다는 것을 알았다. | it = 가목적어, to부정사구 = 진목적어 |

## 출제 포인트

1. `make it + 형용사 + to/that` 패턴에서 it을 빠뜨리지 않는지 확인
2. 가목적어 it이 있을 때 뒤에 진목적어(to부정사/that절) 반드시 동반
