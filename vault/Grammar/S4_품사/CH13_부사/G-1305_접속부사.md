# 접속부사

> [!info] 출처
> 해커스 토익 RC 기본서 (2023) — Section 4 품사 > Chapter 13 부사 > 항목 5 (p.163)

## 핵심 개념

**접속부사**는 앞 문장과 뒤 문장의 논리적 관계(역접, 인과, 추가 등)를 나타내는 부사이다. 접속사와 달리, 접속부사는 **두 독립절을 직접 연결하지 못한다**. 따라서 두 독립절 사이에는 반드시 **세미콜론(;) 또는 마침표**가 와야 한다.

접속부사의 위치:
- **문장 맨 앞**: **However**, the project was delayed.
- **문장 중간 (주어 뒤)**: The project, **however**, was delayed.
- **문장 맨 뒤**: The project was delayed, **however**.

접속부사와 접속사의 결정적 차이:
- 접속사(but, although 등)는 두 절을 하나의 문장으로 연결한다.
- 접속부사(however, therefore 등)는 두 독립 문장을 논리적으로 연결한다.

## 핵심 공식/패턴

```
문장1. [접속부사], 문장2.
문장1; [접속부사], 문장2.
```

| 의미 관계 | 접속부사 | 한국어 의미 |
|-----------|----------|-------------|
| 역접 / 대조 | however | 그러나, 하지만 |
| 역접 / 대조 | nevertheless / nonetheless | 그럼에도 불구하고 |
| 역접 / 대조 | on the other hand | 반면에, 한편으로는 |
| 결과 / 인과 | therefore / thus / hence | 따라서, 그러므로 |
| 결과 / 인과 | consequently / as a result | 결과적으로 |
| 추가 / 첨가 | moreover / furthermore / in addition | 게다가, 더욱이 |
| 추가 / 첨가 | besides / additionally | 또한, 게다가 |
| 예시 | for example / for instance | 예를 들어 |
| 요약 | in conclusion / in summary | 결론적으로, 요약하면 |
| 시간 순서 | meanwhile / in the meantime | 그 동안, 한편 |
| 대안 | otherwise / alternatively | 그렇지 않으면, 또는 |

## 예문

| # | 예문 | 해석 | 포인트 |
|---|------|------|--------|
| 1 | The budget was reduced. **However**, the project continued. | 예산이 삭감되었다. 그러나 프로젝트는 계속되었다. | 역접, 두 독립문장 |
| 2 | Sales increased by 20%; **therefore**, bonuses were paid. | 매출이 20% 증가했다. 따라서 보너스가 지급되었다. | 인과, 세미콜론 사용 |
| 3 | The product is affordable. **Moreover**, it is high quality. | 그 제품은 저렴하다. 게다가 품질도 좋다. | 추가 정보 |
| 4 | Please submit the form by Friday; **otherwise**, your application will not be considered. | 금요일까지 양식을 제출하세요; 그렇지 않으면 지원이 처리되지 않습니다. | 조건/결과 |
| 5 | The meeting was long. **Nevertheless**, all issues were resolved. | 회의가 길었다. 그럼에도 불구하고 모든 문제가 해결되었다. | 양보/역접 |

## 출제 포인트

1. **접속부사 vs 접속사**: 두 절이 이미 각각 독립적이고 사이에 마침표나 세미콜론이 있으면 접속부사, 한 문장 안에서 두 절을 연결하면 접속사를 선택한다.
2. **however**는 접속부사이므로 콤마만으로 두 절을 연결할 수 없다. (`문장1, however, 문장2`는 문법 오류 — comma splice)
3. **therefore**와 **thus**는 의미는 같지만, `thus`는 좀 더 격식체이고 콤마 없이 쓰이기도 한다.
4. **otherwise**는 "그렇지 않으면"의 의미로 조건의 결과를 나타낼 때 자주 쓰인다.
5. Part 6에서 빈칸에 올 접속부사를 고르는 문제는 앞 문장과의 논리 관계를 파악하는 것이 핵심이다.
