# so / such / very / too 구별

> [!info] 출처
> 해커스 토익 RC 기본서 (2023) — Section 4 품사 > Chapter 13 부사 > 항목 7 (p.165)

## 핵심 개념

`so`, `such`, `very`, `too`는 모두 "매우"의 의미를 가지지만, **뒤에 오는 품사와 구조**가 다르기 때문에 구별해야 한다.

핵심 구별 기준:
- **very**: 형용사/부사 앞 (단독 수식, 결과절 없음)
- **so**: 형용사/부사 앞 + (that절 동반 가능)
- **such**: 관사+명사 앞 (a/an + 형용사 + 명사, 또는 명사 앞)
- **too**: 형용사/부사 앞 (부정적 의미 "너무 ~해서 ~할 수 없다" + to부정사)

## 핵심 공식/패턴

```
very + 형용사/부사
so + 형용사/부사 (+ that절)
such + (a/an) + (형용사) + 명사 (+ that절)
too + 형용사/부사 + (to부정사)
```

| 표현 | 구조 | 의미 |
|------|------|------|
| very | very + 형용사/부사 | 매우 (중립적) |
| so | so + 형용사/부사 (that ...) | 매우 ~해서 (결과 강조) |
| such | such + (a/an) + (형용사) + 명사 (that ...) | 매우 대단한 ~이어서 |
| too | too + 형용사/부사 (to do) | 너무 ~해서 ~할 수 없다 (부정적) |

### so vs such 핵심 패턴

| so | such |
|----|------|
| so + 형용사 + (that) | such + a/an + 형용사 + 단수명사 + (that) |
| so + 형용사 + a/an + 명사 | such + 형용사 + 복수명사 |
| so + 부사 + (that) | such + 형용사 + 불가산명사 |

## 예문

| # | 예문 | 해석 | 포인트 |
|---|------|------|--------|
| 1 | The presentation was **very** impressive. | 발표는 매우 인상적이었다. | very + 형용사 |
| 2 | The traffic was **so** heavy that we arrived late. | 교통이 너무 막혀서 늦게 도착했다. | so + 형용사 + that절 |
| 3 | It was **such** a good presentation that everyone applauded. | 너무 훌륭한 발표여서 모두가 박수를 쳤다. | such + a + 형용사 + 명사 |
| 4 | The price was **too** high to accept. | 가격이 너무 높아서 수락할 수 없었다. | too + 형용사 + to부정사 |
| 5 | She ran **so** quickly that she finished first. | 그녀는 너무 빨리 달려서 1등으로 완주했다. | so + 부사 + that절 |
| 6 | They are **such** dedicated employees that they often work overtime. | 그들은 너무 헌신적인 직원들이라 자주 야근을 한다. | such + 형용사 + 복수명사 |
| 7 | The contract was **too** complex to understand quickly. | 계약서가 너무 복잡해서 빨리 이해할 수 없었다. | too ~ to부정사 (부정) |

## 출제 포인트

1. **so vs such**: 뒤에 형용사(+부사)만 오면 `so`, 명사(+수식어)가 오면 `such`를 선택한다.
   - *so + 형용사*: so important
   - *such + a/an + 형용사 + 명사*: such a difficult problem
2. **so + 형용사 + a + 명사** 어순도 가능: `so beautiful a day` (= `such a beautiful day`)
3. **too**는 부정적 결과를 내포하며, `too ~ to do` 구문에서 쓰인다. "너무 ~해서 ~할 수 없다"는 의미.
4. **very**는 `that절`이나 `to부정사`와 결합하지 않는 단순 강조 부사이다.
5. 토익에서 "such a + 형용사 + 명사"와 "so + 형용사 + a + 명사"가 모두 보기에 나올 수 있으므로 둘 다 정문임을 알아야 한다.
