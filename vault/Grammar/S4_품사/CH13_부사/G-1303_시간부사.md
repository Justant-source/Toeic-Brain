# 시간 부사

> [!info] 출처
> 해커스 토익 RC 기본서 (2023) — Section 4 품사 > Chapter 13 부사 > 항목 3 (p.161)

## 핵심 개념

**시간 부사**는 동작이나 상태가 일어나는 시간적 관계를 나타내는 부사이다. 토익에서는 시제와 연결하여 출제되는 경우가 많으므로, 각 시간 부사가 어떤 시제와 주로 어울리는지를 함께 암기해야 한다.

주요 시간 부사와 어울리는 시제:
- **already** (이미, 벌써) → 현재완료, 과거
- **yet** (아직, 이미) → 현재완료 (부정문: "아직 ~않다" / 의문문: "이미 ~했는가")
- **still** (아직도, 여전히) → 현재, 현재완료, 과거진행
- **recently / lately** (최근에) → 현재완료, 과거
- **once** (한때, 이전에) → 과거
- **now / currently / at present** (현재, 지금) → 현재, 현재진행
- **soon / shortly** (곧) → 미래
- **previously / formerly** (이전에) → 과거
- **immediately** (즉시) → 과거, 현재, 미래

## 핵심 공식/패턴

| 시간 부사 | 의미 | 주로 쓰는 시제 |
|-----------|------|----------------|
| already | 이미, 벌써 | 현재완료, 과거 |
| yet | 아직(부정), 이미(의문) | 현재완료 |
| still | 아직도, 여전히 | 현재, 현재완료 |
| recently / lately | 최근에 | 현재완료, 과거 |
| now / currently | 지금, 현재 | 현재, 현재진행 |
| soon / shortly | 곧 | 미래 |
| previously / formerly | 이전에, 예전에 | 과거 |
| immediately | 즉시, 바로 | 과거, 현재, 미래 |
| once | 한때, 과거에 | 과거 |
| eventually | 결국, 마침내 | 과거, 미래 |

```
주어 + have/has + [already / recently / just / yet / still] + p.p.
주어 + [currently / now] + 동사
주어 + will + [soon / shortly] + 동사원형
주어 + [previously / formerly] + 과거동사
```

## 예문

| # | 예문 | 해석 | 포인트 |
|---|------|------|--------|
| 1 | The report has **already** been submitted. | 보고서는 이미 제출되었다. | already + 현재완료 |
| 2 | The package has not arrived **yet**. | 소포가 아직 도착하지 않았다. | yet + 현재완료 부정 |
| 3 | The company is **still** reviewing the proposal. | 회사는 여전히 제안서를 검토 중이다. | still + 현재진행 |
| 4 | **Recently**, the firm has expanded to Asia. | 최근, 그 회사는 아시아로 확장했다. | recently + 현재완료 |
| 5 | She is **currently** working on a new project. | 그녀는 현재 새 프로젝트를 진행 중이다. | currently + 현재진행 |
| 6 | The new policy will take effect **soon**. | 새 정책이 곧 시행될 것이다. | soon + 미래 |
| 7 | He was **previously** employed at XYZ Corp. | 그는 이전에 XYZ 회사에서 일했다. | previously + 과거 |
| 8 | Please respond **immediately** upon receipt. | 수령 즉시 바로 답변해 주세요. | immediately = 즉시 |

## 출제 포인트

1. **already**와 **yet**의 구별: `already`는 긍정문에서 "이미", `yet`은 부정문에서 "아직", 의문문에서 "이미"로 쓰인다.
2. **still**과 **yet**의 구별: `still`은 긍정·부정·의문 모두 가능하고 "아직도"의 뉘앙스, `yet`은 부정·의문에서 사용.
3. **recently**는 현재완료와 과거 모두 가능하지만, `lately`는 현재완료와 더 자주 쓰인다.
4. **currently**와 **now**는 현재시제나 현재진행형과 어울리며, 미래나 과거시제와는 어색하다.
5. **previously / formerly**는 과거 직업, 위치, 이름 등을 언급할 때 주로 사용된다. (*formerly known as*, *previously served as*)
