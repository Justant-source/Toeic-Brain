# 한정사와 명사

> [!info] 출처
> 해커스 토익 RC 기본서 (2023) — Section 4 품사 > Chapter 10 명사 > 항목 3 (p.132)

## 핵심 개념

**한정사(Determiner)**는 명사 앞에 놓여 그 명사를 한정(특정)하는 역할을 한다. 한정사의 종류에 따라 뒤에 올 수 있는 명사의 형태(단수/복수/불가산)가 달라진다.

한정사 종류:
1. **관사**: a/an (부정관사, 가산단수), the (정관사, 모든 명사)
2. **소유격**: my, your, his, her, its, our, their + 명사
3. **지시형용사**: this/that + 단수, these/those + 복수
4. **수량 한정사**: each, every, either, neither + 단수명사 / both, all, several, many + 복수명사 / much, a little + 불가산명사
5. **부정한정사**: some, any, no + 가산복수 또는 불가산

## 핵심 공식/패턴

| 한정사 | 결합 명사 형태 | 예시 |
|--------|--------------|------|
| a / an | 가산단수 | a proposal, an update |
| the | 모든 명사 | the report, the information |
| each / every | 가산단수 | each employee, every item |
| both / all / several / many | 가산복수 | both options, several employees |
| much / a little | 불가산 | much effort, a little time |
| some / any / no | 가산복수·불가산 | some questions, any information |

```
한정사 + (형용사) + 명사
The     new       manager arrived.
Each    individual branch  must report.
Much    additional  information is needed.
```

## 예문

| # | 예문 | 해석 | 포인트 |
|---|------|------|--------|
| 1 | **Each** department must submit **a** report. | 각 부서는 보고서를 제출해야 한다. | each + 가산단수 |
| 2 | **Several** candidates applied for the position. | 몇몇 지원자들이 그 직책에 지원했다. | several + 가산복수 |
| 3 | **Much** attention was given to the details. | 세부 사항에 많은 주의가 기울여졌다. | much + 불가산 |
| 4 | **Both** proposals were reviewed by the committee. | 두 제안 모두 위원회에서 검토됐다. | both + 가산복수 |
| 5 | **Every** employee needs to attend the training. | 모든 직원이 교육에 참석해야 한다. | every + 가산단수 |

## 출제 포인트

1. **each/every + 단수명사**: `each employees`는 틀림. `each employee`가 올바른 표현.
2. **both + 복수명사**: `both option`은 틀림. `both options`가 올바른 표현.
3. **much + 불가산명사**: `much reports`는 틀림. `much information`처럼 불가산명사와 결합.
4. **관사 위치**: 관사는 항상 명사구의 맨 앞(형용사보다도 앞)에 위치한다. `a new policy` (O) / `new a policy` (X)
5. **소유격과 관사 동시 사용 불가**: `the my report` (X), `my report` (O) 또는 `the report` (O).
