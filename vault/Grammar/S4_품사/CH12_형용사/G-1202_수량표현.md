# 수량 표현 형용사 — many / much / few / little / a number of / the number of

> [!info] 출처
> 해커스 토익 RC 기본서 (2023) — Section 4 품사 > Chapter 12 형용사 > 항목 2 (p.150)

## 핵심 개념

수량을 나타내는 형용사(수량 형용사)는 수식하는 명사가 **가산명사**인지 **불가산명사**인지에 따라 구별해야 한다.

핵심 원리:
- **가산명사**: many, few, a few, several, a number of, the number of
- **불가산명사**: much, little, a little, a great deal of, the amount of
- **가산·불가산 모두**: some, any, no, a lot of, lots of, plenty of, most, all

수량 표현이 주어 역할을 할 때, **수 일치**도 함께 중요하다:
- `a number of + 복수명사` → 복수 동사
- `the number of + 복수명사` → 단수 동사

## 핵심 공식/패턴

### 가산 vs 불가산 수량 표현

| 수량 표현 | 가산명사 | 불가산명사 | 의미 |
|-----------|---------|-----------|------|
| many | O | X | 많은 |
| much | X | O | 많은 |
| few | O | X | 거의 없는 (부정) |
| a few | O | X | 몇몇의 (긍정) |
| little | X | O | 거의 없는 (부정) |
| a little | X | O | 약간의 (긍정) |
| several | O | X | 여러 개의 |
| a number of | O | X | 많은 |
| the number of | O | X | ~의 수 |
| a great deal of | X | O | 많은 양의 |
| the amount of | X | O | ~의 양 |
| a lot of / lots of | O | O | 많은 |
| some / any | O | O | 몇몇의 / 어떤 |
| most / all | O | O | 대부분 / 모든 |

### few vs a few / little vs a little

```
few    = 거의 없는  (부정적)   a few   = 몇몇의 (긍정적)
little = 거의 없는  (부정적)   a little = 약간의 (긍정적)

few employees attended  (직원이 거의 없었다 — 부정적 어감)
a few employees attended (직원 몇 명이 참석했다 — 긍정적 어감)
```

### a number of vs the number of — 수 일치

```
a number of + 복수명사 → 복수 동사  ("많은 ~")
A number of complaints HAVE been received.

the number of + 복수명사 → 단수 동사  ("~의 수")
The number of complaints HAS increased.
```

## 예문

| # | 예문 | 해석 | 포인트 |
|---|------|------|--------|
| 1 | **Many** employees attended the company picnic. | 많은 직원들이 회사 소풍에 참석했다. | many + 가산 복수 |
| 2 | We don't have **much** time before the deadline. | 마감 전까지 시간이 많지 않다. | much + 불가산 |
| 3 | There are **a few** options available for you. | 몇 가지 선택지가 있습니다. | a few = 긍정 (몇몇) |
| 4 | There is **little** evidence to support the claim. | 그 주장을 뒷받침할 증거가 거의 없다. | little + 불가산 (부정) |
| 5 | **A number of** new policies have been implemented. | 많은 새 정책들이 시행되었다. | a number of → 복수 동사 |
| 6 | **The number of** online orders has doubled this year. | 온라인 주문 건수가 올해 두 배로 늘었다. | the number of → 단수 동사 |
| 7 | We've received **several** inquiries about the product. | 제품에 대해 여러 건의 문의를 받았다. | several + 가산 복수 |

## 출제 포인트

> [!warning] 토익 출제 포인트
> **가산/불가산 명사와 수량 표현의 결합, 그리고 a number of vs the number of의 수 일치가 핵심 출제 포인트이다.**
>
> 1. **many vs much**: 빈칸 뒤 명사가 복수이면 `many`, 불가산이면 `much`
>    - *___ feedback* → **much** (feedback은 불가산)
>    - *___ suggestions* → **many** (suggestions는 가산 복수)
> 2. **few vs a few**: `few`는 "거의 없어서 문제", `a few`는 "조금 있어서 괜찮다"
> 3. **a number of** → 복수 취급, **the number of** → 단수 취급
> 4. **a lot of**는 가산·불가산 모두 사용 가능하므로 안전한 선택
