# 전치사+관계대명사 / 수량 표현+관계대명사 / 관계대명사 생략

> [!info] 출처
> 해커스 토익 RC 기본서 (2023) — Section 5 접속사와절 > Chapter 16 관계절 > 항목 3 (p.193)

## 핵심 개념

### 전치사 + 관계대명사

앞 문장과의 공통 명사가 뒷 문장에서 **전치사의 목적어**일 때 `[전치사 + 관계대명사]` 형태가 된다.

변환 과정:
> I interviewed a woman. + I had previously met with **her**.
> → I interviewed a woman **whom** I had previously met with.
> → I interviewed a woman **with whom** I had previously met.

- 전치사의 위치: 전치사는 **선행사** 또는 **관계절 내 동사**에 따라 결정
- `[전치사 + 관계대명사]` 뒤에는 **완전한 절**이 옴 (목적어가 없는 불완전한 절 아님)
- `[전치사 + 관계대명사]` 형태에서는 **that 사용 불가**

### 수량 표현 + 관계대명사

```
one/each/some/any/all/both/several/many/much/most/half/the rest + of + whom/which/whose + 명사
```

> The class has 20 students, **all of whom** are boys.
> (= The class has 20 students, and all of them are boys.)

### 관계대명사의 생략

**① 주격 관계대명사 + be동사 생략**
- `(which was)` 생략 → 분사/형용사/전치사구가 남음
  - `The location **(which was)** selected for construction is downtown.`
  - `I need to speak with the person **(who is)** responsible for reservations.`
  - `The parking lot **(which is)** near our office has been closed.`

**② 목적격 관계대명사 생략**
- `(whom/who/which)` 생략 → `주어 + 동사`가 남음
  - `Laura Secord is the woman **(whom)** I called yesterday.`
  - `This is the form **(which)** you must fill out for membership.`

## 핵심 공식/패턴

```
전치사 + whom/which + 완전한 절
(동사 뒤 전치사 이동 → 완전한 절이 됨)

수량표현 + of + whom(사람)/which(사물) + 동사

주격 관계대명사 + be동사 → 생략 가능 → 분사/형용사/전치사구 남음
목적격 관계대명사 → 생략 가능 → 주어 + 동사 남음
```

## 예문

| # | 예문 | 해석 | 포인트 |
|---|------|------|--------|
| 1 | I will call the client to **whom** the package was delivered. | 소포가 배송된 고객에게 전화할 것이다. | 전치사 + whom |
| 2 | This is the software **about which** Mr. Lawson talked. | 이것이 Lawson이 이야기했던 소프트웨어이다. | talk about → about which |
| 3 | The class has 20 students, **all of whom** are boys. | 그 반에는 20명이 있는데, 그들 모두가 남학생이다. | 수량 표현 of whom |
| 4 | The location **(which was)** selected is downtown. | 건설을 위해 선정된 장소는 도심지이다. | 주격+be 생략 → 분사 |
| 5 | Laura Secord is the woman **(whom)** I called. | Laura Secord는 내가 전화했던 여자이다. | 목적격 생략 |

## 출제 포인트

1. **전치사 + 관계대명사 채우기**: 전치사 뒤에 whom/which 선택, that 불가
2. **수량 표현 + of + whom/which**: 선행사가 사람이면 whom, 사물이면 which
3. **관계대명사 + be동사 생략 후 형태**: 분사, 형용사, 전치사구 채우기
4. **목적격 생략 후**: 주어 + 동사 형태가 남음
