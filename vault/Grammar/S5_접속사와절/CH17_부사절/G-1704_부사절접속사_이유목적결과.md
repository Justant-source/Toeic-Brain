# 부사절 접속사: 이유·목적·결과

> [!info] 출처
> 해커스 토익 RC 기본서 (2023) — Section 5 접속사와절 > Chapter 17 부사절 > 항목 4 (p.198)

## 핵심 개념

### 이유 부사절 접속사

| 접속사 | 의미 | 특징 |
|-------|------|------|
| **because** | ~이기 때문에 | 가장 직접적인 이유, 주절 뒤·앞 모두 가능 |
| **since** | ~이기 때문에 | 이미 알려진 이유, 주로 주절 앞 |
| **as** | ~이기 때문에 | 부수적·배경 이유, 주로 주절 앞 |
| **now that** | 이제 ~이므로 | 새롭게 생긴 상황의 이유 |

### 목적 부사절 접속사

| 접속사 | 의미 |
|-------|------|
| **so that** | ~하기 위해서, ~할 수 있도록 |
| **in order that** | ~하기 위해서 (격식) |

**so that** 절에는 주로 **조동사(can, will, may, could, would)** 가 온다.
- `She left early so that she could catch the first train.`

### 결과 부사절 접속사

| 표현 | 의미 |
|-----|------|
| **so + 형용사/부사 + that** | 너무 ~해서 …하다 |
| **such + (a/an) + 형용사 + 명사 + that** | 너무 ~한 명사라서 …하다 |

```
so + adj/adv + that + 절
such + (a/an) + adj + 명사 + that + 절
```

### because vs because of 구별

| 형태 | 뒤에 오는 것 | 예시 |
|-----|------------|------|
| **because** (접속사) | 주어 + 동사 (절) | because the weather was bad |
| **because of** (전치사) | 명사(구) | because of the bad weather |

## 핵심 공식/패턴

```
이유: because / since / as / now that + 주어 + 동사

목적: so that + 주어 + 조동사(can/will/may) + 동사원형
      in order that + 주어 + 조동사 + 동사원형

결과: so + 형용사/부사 + that + 절
      such (a/an) + 형용사 + 명사 + that + 절
```

## 예문

| # | 예문 | 해석 | 포인트 |
|---|------|------|--------|
| 1 | **Because** the conference was canceled, we rescheduled the meeting. | 회의가 취소되었기 때문에 미팅 일정을 변경했다. | 직접 이유 |
| 2 | **Since** the office is closed today, no one will answer the phone. | 오늘 사무실이 닫혔으므로 아무도 전화를 받지 않을 것이다. | 알려진 이유 |
| 3 | **Now that** the project is complete, we can celebrate. | 이제 프로젝트가 완료되었으니 축하할 수 있다. | 새로운 상황 |
| 4 | She spoke slowly **so that** everyone **could** understand. | 모두가 이해할 수 있도록 천천히 말했다. | 목적, 조동사 필수 |
| 5 | The product was **so** popular **that** it sold out immediately. | 제품이 너무 인기 있어서 즉시 품절되었다. | so ... that |
| 6 | It was **such** a successful event **that** they plan to repeat it. | 너무 성공적인 행사라서 반복할 계획이다. | such ... that |

## 출제 포인트

1. **so that + 조동사**: so that 뒤에 동사원형만 오면 오답 (조동사 필요)
2. **so vs such**: 형용사/부사 → so, 명사구 → such (a/an)
3. **because vs because of**: 절 앞 → because, 명사구 앞 → because of
4. **now that**: '이제 ~이므로' — 이유 접속사로 출제
