# 부사절 접속사: 조건·양보

> [!info] 출처
> 해커스 토익 RC 기본서 (2023) — Section 5 접속사와절 > Chapter 17 부사절 > 항목 3 (p.198)

## 핵심 개념

### 조건 부사절 접속사

| 접속사 | 의미 |
|-------|------|
| **if** | 만약 ~이라면 |
| **unless** | 만약 ~이 아니라면 (= if ... not) |
| **as long as** | ~하는 한, ~하는 한에서는 |
| **provided (that)** | ~이라면, ~을 조건으로 |
| **only if** | ~하는 경우에만 |
| **once** | 일단 ~하면 (조건) |

### 양보 부사절 접속사

| 접속사 | 의미 |
|-------|------|
| **although / though / even though** | 비록 ~이지만 |
| **even if** | 비록 ~이라 할지라도 |
| **while / whereas** | ~인 반면에 (대조) |

### 조건절의 시제 규칙

조건 부사절(if, unless, as long as, provided that 등) 뒤에서는 **미래를 나타내기 위해 현재 시제**를 사용한다.

- `If the deal (is / will be) finalized, both companies will benefit.` → **is** (조건절 → 현재)

### although vs even if

- **although / even though**: 이미 사실인 내용을 인정하고 그와 대조적인 결과를 말할 때
  - `Although the project was delayed, the client was satisfied.`
- **even if**: 가정의 상황을 나타낼 때 (사실 여부 불확실)
  - `Even if it rains tomorrow, the event will be held outdoors.`

### unless = if ... not

- `Unless you have a reservation, you cannot enter.`
  = `If you do not have a reservation, you cannot enter.`
- **주의**: unless 뒤에는 부정어(not)를 쓰지 않는다.
  - `Unless you don't come (→ Unless you come / If you don't come)`

## 핵심 공식/패턴

```
조건 부사절 + 현재 시제 → 주절: 미래 시제
if / unless / as long as / provided (that) + 현재 동사

양보: although / even though + 절 (사실)
     even if + 절 (가정)

unless = if ... not (이중 부정 불가)
```

## 예문

| # | 예문 | 해석 | 포인트 |
|---|------|------|--------|
| 1 | **If** the shipment **arrives** on time, we can meet the deadline. | 선적품이 제때 도착한다면 마감일을 맞출 수 있다. | 조건절→현재 시제 |
| 2 | **Unless** you submit the form by Friday, your application will be rejected. | 금요일까지 양식을 제출하지 않으면 신청이 거절된다. | unless = if not |
| 3 | You may use the facility **as long as** you are a member. | 회원인 한 시설을 이용할 수 있다. | as long as = ~하는 한 |
| 4 | **Although** the price was high, the product sold well. | 가격이 높았지만 제품은 잘 팔렸다. | 양보 (사실) |
| 5 | **Even if** we increase the budget, the timeline won't change. | 예산을 늘리더라도 일정은 바뀌지 않을 것이다. | 양보 (가정) |
| 6 | **Provided that** all conditions are met, the contract will be signed. | 모든 조건이 충족된다면 계약이 체결될 것이다. | provided that = 조건 |

## 출제 포인트

1. **조건절 + 현재 시제**: if/unless/as long as 뒤에 will 쓰면 오답
2. **unless + 긍정**: unless 뒤에 not을 함께 쓰지 않는다
3. **although vs even if**: 사실 인정(although) vs 가정(even if) 구별
4. **as long as / provided that**: 조건의 의미로 출제 빈도 높음
