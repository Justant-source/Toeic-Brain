# 조동사 + 동사원형

> [!info] 출처
> 해커스 토익 RC 기본서 (2023) — Section 2 동사구 > Chapter 3 동사의 형태와 종류 > 항목 1 (p.69)

## 핵심 개념

조동사(will/would, may/might, can/could, must, should) 다음에는 반드시 **동사원형**이 와야 한다. to 부정사, -ing형, 명사, 3인칭 단수형 동사는 올 수 없다.

다음과 같이 조동사처럼 쓰이는 표현들 뒤에도 동사원형을 써야 한다:
- ought to, have to, had better, be able to, would like to, be going to, used to

부정문이나 도치 구문을 만드는 조동사(do/does/did) 다음에도 반드시 동사원형이 와야 한다.

조동사와 동사원형 사이에 not이나 부사가 끼어들어도 동사원형 자리에 다른 형태를 쓰지 않도록 주의한다.

## 핵심 공식/패턴

```
조동사 + [동사원형]

will / would / may / might / can / could / must / should + 동사원형
ought to / have to / had better / be able to / would like to / be going to / used to + 동사원형
do / does / did (부정문·도치) + 동사원형
```

| 잘못된 형태 | 올바른 형태 | 이유 |
|------------|------------|------|
| will **to answer** | will **answer** | 조동사 뒤 to부정사 불가 |
| can **finishing** | can **finish** | 조동사 뒤 -ing 불가 |
| should **removal** | should **remove** | 조동사 뒤 명사 불가 |
| will **buys** | will **buy** | 조동사 뒤 3인칭 단수형 불가 |
| must not **selects** | must not **select** | 조동사+not 사이에도 동사원형 |
| did not **left** | did not **leave** | 부정문 조동사(did) 뒤 동사원형 |

## 예문

| # | 예문 | 해석 | 포인트 |
|---|------|------|--------|
| 1 | Anne will **answer** the phone. | Anne은 전화를 받을 것이다. | will + 동사원형 (to answer ✗) |
| 2 | Robert can **finish** the marketing report by Monday. | Robert는 월요일까지 보고서를 끝낼 수 있다. | can + 동사원형 (finishing ✗) |
| 3 | Campers should **remove** all trash from the site. | 야영객들은 모든 쓰레기를 치워야 한다. | should + 동사원형 (removal ✗) |
| 4 | Sandra is able to **speak** German fluently. | Sandra는 독일어를 유창하게 말할 수 있다. | be able to + 동사원형 |
| 5 | Never did he **agree** to the moving plan. | 그는 이사 계획에 결코 동의하지 않았다. | 도치 did + 동사원형 (agreed ✗) |
| 6 | The board of directors must not hastily **select** a new CEO. | 이사회는 새로운 CEO를 성급하게 선발해서는 안 된다. | 조동사+not+부사 뒤에도 동사원형 |

## 출제 포인트

- **주요 출제 유형**: 조동사 바로 뒤 빈칸에 동사 형태를 채우는 문제. 보기에 to부정사, -ing형, 명사, 3인칭 단수형 동사를 오답으로 배치한다.
- **함정**: 조동사와 동사원형 사이에 `not`이나 부사(hastily, clearly 등)가 끼어들어도 동사 자리에는 여전히 동사원형이 와야 한다.
- **구별 주의**: "가지다/먹다"를 뜻하는 `have`는 일반동사이므로, 뒤에 p.p.가 오지 않는다. (The library has an extensive encyclopedia section. → 이때 `has`는 조동사가 아닌 일반동사)
