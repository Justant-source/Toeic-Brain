# 혼동하기 쉬운 자동사와 타동사, 3형식과 4형식

> [!info] 출처
> 해커스 토익 RC 기본서 (2023) — Section 2 동사구 > Chapter 3 동사의 형태와 종류 > 항목 3 (p.71)

## 핵심 개념

자동사가 목적어를 갖기 위해서는 반드시 전치사가 있어야 하지만, **타동사는 전치사 없이 목적어를 바로** 가져야 한다.

**3형식 동사**: 목적어(that절)를 하나만 가지는 동사
**4형식 동사**: 목적어를 2개(간접목적어 + 직접목적어)로 가지는 동사

## 핵심 공식/패턴

| 의미 | 자동사 + 전치사 + 목적어 | 타동사 + 목적어 |
|------|----------------------|----------------|
| 말하다 | speak **to** a group / talk **to** your customer / account **for** your absence | mention his absence / discuss the issue / address him |
| 답하다 | reply **to** letters / respond **to** a question | answer the question |
| 동의/반대 | agree **with/to/on** the policy / object **to** the plan / consent **to** a suggestion | approve the request / oppose the new system |
| 기타 | participate **in** the workshop / arrive **at** the airport / wait **for** the new product | attend a seminar / reach the destination / await the release |

**3형식 vs 4형식 구별**:
```
3형식 동사 + 목적어(that절)
  say / mention / announce (to me) that
  suggest / propose / recommend (to me) that
  explain / describe (to me) that

4형식 동사 + 목적어(-에게) + 목적어(that절)
  tell / inform / notify me that
  assure / convince me that
```

## 예문

| # | 예문 | 해석 | 포인트 |
|---|------|------|--------|
| 1 | The director was requested to **account for** the unexpected decrease in sales. | 이사는 예상치 못한 매출 감소에 대해 설명하도록 요구되었다. | 자동사 account + for + 목적어 (explain ✗) |
| 2 | The consultant **told** the owner that operating costs were too high. | 컨설턴트는 소유주에게 운영비가 너무 높다고 말했다. | 4형식 told (said ✗: said는 3형식) |
| 3 | During the presentation, the manager will **talk about** the need to target consumers. | 발표 중 관리자는 소비자 타겟팅의 필요성에 대해 이야기할 것이다. | talk + about (전치사 필요) |
| 4 | An administrative assistant **informed** participants that the venue had been changed. | 행정 보조원은 참가자들에게 장소가 변경되었다고 알렸다. | 4형식: inform + 사람 + that절 |

## 출제 포인트

- **주요 출제 유형**: 4개의 동사 보기 중 문장 구조에 가장 적합한 동사를 선택하는 문제가 출제된다.
- **핵심 판별 기준**: 동사 뒤에 전치사가 있으면 자동사, 전치사 없이 목적어가 바로 오면 타동사가 와야 한다.
- **3형식 vs 4형식 구별**: 목적어가 2개(사람 + that절)이면 4형식 동사(tell, inform, notify 등), 목적어가 1개(that절)이면 3형식 동사(say, mention, explain 등)를 써야 한다.
- **자주 혼동하는 쌍**: discuss ↔ talk about / attend ↔ participate in / reach ↔ arrive at / answer ↔ respond to
