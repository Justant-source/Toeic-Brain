# if 없는 가정법: 도치와 without 구문

> [!info] 출처
> 해커스 토익 RC 기본서 (2023) — Section 2 동사구 > Chapter 6 시제와 가정법 > 항목 6 (p.92)

## 핵심 개념

가정법에서 if를 생략하고 **도치**를 사용하거나, **without/but for** 등으로 가정의 의미를 나타낼 수 있다.

**if 생략 + 도치**:
- 가정법 과거: `Were + 주어 ~` (= If + 주어 + were ~)
  - **Were I** you, I would accept the offer. (= If I were you)
- 가정법 과거완료: `Had + 주어 + p.p. ~` (= If + 주어 + had p.p. ~)
  - **Had she known**, she would have acted differently. (= If she had known)
- 가정법 미래: `Should + 주어 + 동사원형 ~` (= If + 주어 + should + 동사원형)
  - **Should you** need help, call us. (= If you should need help)

**without / but for + 명사**:
- without = if it were not for (현재) / if it had not been for (과거)
- **Without** your support, we could not succeed. (= If it were not for your support)
- **Without** his advice, she would have failed. (= If it had not been for his advice)

## 핵심 공식/패턴

```
Were + S + ..., S + would/could + 동사원형
Had + S + p.p. ..., S + would/could + have p.p.
Should + S + 동사원형, S + would/will + 동사원형
Without + 명사 = if it were/had not been for + 명사
```

## 예문

| # | 예문 | 해석 | 포인트 |
|---|------|------|--------|
| 1 | **Had the manager approved** the budget, we could have hired more staff. | 관리자가 예산을 승인했더라면 더 많은 직원을 채용할 수 있었을 텐데. | Had + 주어 + p.p. → if 생략 도치 |
| 2 | **Without** the government subsidy, the project would have been cancelled. | 정부 보조금이 없었다면 프로젝트가 취소되었을 것이다. | without = if it had not been for |

## 출제 포인트

1. 문장이 `Were/Had/Should`로 시작 → if가 생략된 가정법 도치
2. `without` + 명사 뒤에 주절에 `would/could + 동사원형` or `would have p.p.` → 가정법
3. `but for` = `without` = `if it were not for` (동의 표현)
