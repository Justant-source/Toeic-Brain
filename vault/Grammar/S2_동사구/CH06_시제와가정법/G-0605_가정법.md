# 가정법 미래/과거/과거완료

> [!info] 출처
> 해커스 토익 RC 기본서 (2023) — Section 2 동사구 > Chapter 6 시제와 가정법 > 항목 5 (p.92)

## 핵심 개념

가정법은 현실과 반대되는 상황을 가정하거나 실현 가능성이 낮은 상황을 나타낼 때 쓴다. 시제를 한 단계 앞으로 당겨 쓰는 것이 특징이다.

**가정법 과거** (현재 사실의 반대):
- `If + 주어 + 과거동사, 주어 + would/could/might + 동사원형`
- If I **were** you, I would accept the offer.
- be동사는 인칭과 관계없이 **were**를 쓴다.

**가정법 과거완료** (과거 사실의 반대):
- `If + 주어 + had p.p., 주어 + would/could/might + have p.p.`
- If she **had applied** earlier, she would have gotten the job.

**가정법 미래** (미래에 실현 가능성이 낮음):
- `If + 주어 + should + 동사원형, 주어 + would/will + 동사원형`
- If you **should** need assistance, please contact us.

## 핵심 공식/패턴

```
가정법 과거:      If + S + 과거동사(were), S + would/could + 동사원형
가정법 과거완료:  If + S + had p.p., S + would/could + have p.p.
가정법 미래:      If + S + should + 동사원형, S + would/will + 동사원형
```

## 예문

| # | 예문 | 해석 | 포인트 |
|---|------|------|--------|
| 1 | If the budget **were** larger, we **could hire** more staff. | 예산이 더 많다면 직원을 더 채용할 수 있을 텐데. | 가정법 과거: 현재 사실 반대 |
| 2 | If we **had started** earlier, we **would have met** the deadline. | 더 일찍 시작했더라면 마감을 맞췄을 텐데. | 가정법 과거완료: 과거 사실 반대 |

## 출제 포인트

1. `If + 과거동사 ~ , would + 동사원형` → 가정법 과거 (현재 반대)
2. `If + had p.p. ~ , would have p.p.` → 가정법 과거완료 (과거 반대)
3. `If I were you` → were (was 사용 불가)
