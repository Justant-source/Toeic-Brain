# 수동태 동사 숙어

> [!info] 출처
> 해커스 토익 RC 기본서 (2023) — Section 2 동사구 > Chapter 5 능동태·수동태 > 항목 4 (p.84)

## 핵심 개념

수동태 형태로 굳어진 표현들로, 전치사와 함께 하나의 숙어처럼 쓰인다. 토익에서 전치사 선택 문제로 자주 출제된다.

| 표현 | 의미 |
|------|------|
| be associated **with** | ~와 연관되다 |
| be committed **to** + 명사/동명사 | ~에 전념하다 |
| be dedicated **to** + 명사/동명사 | ~에 헌신하다 |
| be qualified **for** | ~의 자격이 있다 |
| be responsible **for** | ~에 책임이 있다 |
| be known **for** | ~로 알려지다 (이유) |
| be known **as** | ~로 알려지다 (자격/역할) |
| be located **in/at** | ~에 위치하다 |
| be pleased/satisfied/delighted **with** | ~에 만족하다 |
| be equipped **with** | ~을 갖추다 |
| be involved **in** | ~에 관여하다 |
| be based **in/on** | ~에 기반하다 |
| be scheduled **for/to** | ~로 예정되다 |
| be required **to** | ~하도록 요구받다 |

## 핵심 공식/패턴

```
be + p.p. + 전치사 + 명사(구)
전치사 선택은 숙어로 암기
```

## 예문

| # | 예문 | 해석 | 포인트 |
|---|------|------|--------|
| 1 | The company is **committed to** providing quality service. | 회사는 양질의 서비스 제공에 전념하고 있다. | committed to + 동명사 |
| 2 | She is **responsible for** managing the budget. | 그녀는 예산 관리를 담당하고 있다. | responsible for + 명사/동명사 |

## 출제 포인트

1. `be committed to + 동명사` — to는 전치사이므로 뒤에 동명사(V-ing) 사용
2. `be known for` (이유) vs `be known as` (역할/자격) 구별
3. `be located in` (도시/지역) vs `be located at` (구체적 주소/장소)
