# 원급 비교

> [!info] 출처
> 해커스 토익 RC 기본서 (2023) — Section 6 특수구문 > Chapter 19 비교구문 > 항목 1 (p.220)

## 핵심 개념

원급 비교는 **두 대상이 같은 정도임**을 나타낸다. `as + 형용사/부사 원급 + as` 구조를 사용한다.

### 기본 구조

```
as + 형용사/부사 원급 + as
```

- `This hotel is **as luxurious as** the one downtown.` (두 호텔이 동등하게 럭셔리하다)
- `She works **as efficiently as** her colleagues.` (동등하게 효율적으로)

### 부정 원급 비교

- `not as/so + 형용사/부사 + as` → ~만큼 ~하지 않다
- `This product is **not as durable as** the previous model.`

### 원급 비교 관련 표현

| 표현 | 의미 |
|-----|------|
| **as + 원급 + as possible** | 가능한 한 ~하게 |
| **as + 원급 + as + 주어 + can** | 가능한 한 ~하게 |
| **twice as + 원급 + as** | ~의 두 배만큼 ~한 |
| **half as + 원급 + as** | ~의 절반만큼 ~한 |
| **the same + 명사 + as** | ~와 같은 |

### as ~ as 사이에 오는 것

as와 as 사이에는 **형용사 또는 부사**가 온다. 명사가 들어갈 수 없다.

- `as **quickly** as possible` (부사 ✓)
- `as **important** as ever` (형용사 ✓)

## 핵심 공식/패턴

```
동등 비교:   as + 형용사/부사 원급 + as
부정 비교:   not as/so + 형용사/부사 원급 + as

배수 표현:   twice / three times as + 원급 + as
가능한 한:   as + 원급 + as possible
             as + 원급 + as + 주어 + can
```

## 예문

| # | 예문 | 해석 | 포인트 |
|---|------|------|--------|
| 1 | The new branch is **as large as** the main office. | 새 지점은 본사만큼 크다. | 동등 비교 |
| 2 | Please respond **as soon as possible**. | 가능한 한 빨리 응답해 주세요. | as ... as possible |
| 3 | Our new model is **twice as fast as** the old one. | 신형 모델은 구형의 두 배 빠르다. | 배수 표현 |
| 4 | The annual report is **not as comprehensive as** last year's. | 연간 보고서가 작년 것만큼 포괄적이지 않다. | 부정 원급 |
| 5 | The delivery arrived **as early as** expected. | 배송이 예상만큼 일찍 도착했다. | 부사 비교 |

## 출제 포인트

1. **as ~ as 사이 원급**: 비교급/최상급이 들어오면 오답
2. **배수 표현 어순**: twice **as + 원급 + as** (twice more than 형태 주의)
3. **as ~ as possible vs as ~ as can**: 주어가 사람이면 둘 다 가능
4. **형용사 vs 부사**: as와 as 사이의 품사 확인 (명사 수식 → 형용사, 동사/형용사 수식 → 부사)
