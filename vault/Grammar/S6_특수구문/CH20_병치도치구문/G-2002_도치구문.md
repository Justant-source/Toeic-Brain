# 도치구문

> [!info] 출처
> 해커스 토익 RC 기본서 (2023) — Section 6 특수구문 > Chapter 20 병치도치구문 > 항목 2 (p.228)

## 핵심 개념

도치구문은 **주어와 동사의 어순이 바뀌는** 구문이다. 특정 부사(구)가 문두에 오면 주어와 동사가 도치된다.

### 도치가 일어나는 경우

**1. 부정어(구)가 문두에 올 때**

부정적 의미의 부사(구)가 강조를 위해 문두에 오면, **조동사 + 주어 + 동사원형** 어순이 된다.

| 부정어/부정 부사구 | 예시 |
|-----------------|------|
| **Never** | Never have I seen such a result. |
| **Rarely / Seldom / Hardly / Scarcely** | Rarely does he miss a deadline. |
| **Not only** | Not only did she finish on time, but she also exceeded expectations. |
| **Not until** | Not until the project ended did we celebrate. |
| **Under no circumstances** | Under no circumstances should you share the password. |
| **In no way** | In no way does this affect our decision. |
| **Only + 부사구** | Only then did we realize the error. |
| **Only after/when/if + 절** | Only after the meeting ended did she leave. |

**도치 형태:**
```
부정어(구) + 조동사 + 주어 + 동사원형
부정어(구) + be동사 + 주어
부정어(구) + have/has/had + 주어 + p.p. (완료형)
```

**2. only + 부사(구/절)가 문두에 올 때**

- `**Only by** working together **can** we achieve our goals.`
- `**Only when** the manager approves **will** the project proceed.`

**3. so / neither / nor가 문두에 올 때 (동의/부정 표현)**

- `A: "I enjoyed the presentation."` → `B: "**So did** I."` (나도 그랬다)
- `A: "I couldn't attend."` → `B: "**Neither could** I."` / `"**Nor could** I."` (나도 못 했다)

**4. 조건절에서 if 생략 시 도치**

if가 생략되면 **주어와 조동사/be동사가 도치**된다.

| 일반형 | if 생략 도치형 |
|------|-------------|
| If I **were** you | **Were** I you |
| If he **had** known | **Had** he known |
| If this **should** happen | **Should** this happen |

이 구문은 가정법과 밀접하게 연결된다. (→ [[G-0606_if없는가정법]])

## 핵심 공식/패턴

```
부정어 도치: 부정어(구) + 조동사/be동사 + 주어 + (동사원형/p.p.)
  Never, Rarely, Seldom, Hardly, Scarcely
  Not only, Not until, Under no circumstances, In no way
  Only + 부사(구/절)

So/Neither/Nor 도치:
  So + 조동사 + 주어  (긍정 동의)
  Neither/Nor + 조동사 + 주어  (부정 동의)

if 생략 도치:
  Were + 주어 + ... (= If + 주어 + were ...)
  Had + 주어 + p.p. (= If + 주어 + had + p.p.)
  Should + 주어 + V (= If + 주어 + should + V)
```

## 예문

| # | 예문 | 해석 | 포인트 |
|---|------|------|--------|
| 1 | **Never** has the company faced such a challenge. | 회사가 이런 도전에 직면한 적이 없다. | Never + have + 주어 + p.p. |
| 2 | **Rarely does** the CEO attend these meetings. | CEO가 이 회의에 참석하는 경우는 드물다. | Rarely + does + 주어 + 동사원형 |
| 3 | **Not only did** the team meet the deadline, but they also cut costs. | 팀은 기한을 맞췄을 뿐만 아니라 비용도 절감했다. | Not only + did + 주어 + V |
| 4 | **Only after** the audit was completed **did** the manager sign the report. | 감사가 완료된 후에야 매니저가 보고서에 서명했다. | Only after + 절 + did + 주어 |
| 5 | **So do** I. / **Neither do** I. | 나도 그렇다. / 나도 아니다. | So/Neither 도치 |
| 6 | **Should** you need any assistance, please call us. | 도움이 필요하시면 전화해 주세요. | Should 도치 = If you should need |
| 7 | **Had** the shipment arrived on time, we would have met the deadline. | 선적품이 제때 도착했더라면 마감일을 맞출 수 있었을 것이다. | Had 도치 = If had |

## 출제 포인트

1. **부정어 문두 → 도치**: Never/Rarely/Seldom 등이 문두에 오면 조동사+주어 도치
2. **Only + 부사구 문두 → 도치**: Only by/when/after 등도 도치 유발
3. **if 생략 도치**: Were/Had/Should로 시작하는 문장은 if 생략 가정법
4. **So/Neither 도치**: 앞 문장이 긍정이면 So, 부정이면 Neither/Nor
