## 처리 방식

### Phase 1 — 규칙 기반 자동 수정 (18,183개)
- `斤` → `fr` 치환 (斤om→from, 斤ee→free, 斤ozen→frozen 등)
- 한글 이후 텍스트 잘라내기 (한글+영문 혼재 문장에서 영어 부분만 추출)
- UI 기호 제거 (▼▲☐☑✓✔ 등)
- 선 문자 제거 (─│├┤═║ 등)
- 연속 공백 정규화

### Phase 2 — AI 멀티 에이전트 재구성 (1,713개)
- 규칙으로 해결 안 되는 22개 배치 파일 생성 (배치당 80개 문장)
- 4개 에이전트 병렬 실행 (batches 1-6, 7-12, 13-17, 18-22)
- 주요 복원 패턴:
  - `M이iday's` → `Monday's` (한글 자모 삽입 복원)
  - `Itoemphasized` → `Ito emphasized` (공백 누락 복원)
  - `G아izalez` → `Gonzalez` (고유명사 복원)
  - 이메일 헤더 `=====*E-mail*===` 노이즈 정리
  - 표/양식 데이터에서 영어 텍스트 추출

### Phase 3 — 잔여 직접 수정 (25개)
- `Now 7/ome` 패턴 → `Now Home` (주문서 데이터)
- 수학 기호 완전 깨진 문장 → 빈 문자열 처리
