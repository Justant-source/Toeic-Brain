## 작업 1: Part5 해설 화면 - 정답 완성 문장 수정

### 문제 원인
`make_filled_sentence()` 함수가 `-------`(7개 대시) 패턴만 처리하여, 다른 빈칸 패턴(em dash, 점선 등)이나 빈칸 마커가 없는 문장에서 정답 삽입이 안 됨.

### 수정 내용

**파일**: `scripts/anki/generate_part5_deck.py` (line 120-155)

| 항목 | 수정 전 | 수정 후 |
|------|---------|---------|
| 빈칸 패턴 | `-------` 1종만 | 5종 (`-------`, `—■`, `——`, `........`, `—`) |
| 소유격 `^` | 미처리 | `^s` → `'s`, `^ ` → `'s ` 변환 |
| 빈칸 없는 문장 | 원문 그대로 출력 | `[→ <b>정답</b>]` 형태로 fallback |

### 데이터 분석 결과
- 총 1,467문제 중:
  - `-------` 패턴: 1,420문제 (96.8%) — 기존 코드 정상 처리
  - 대안 빈칸 패턴 (`—■`, `——`, `—`, `........`): 5문제
  - 빈칸 마커 없음 (OCR 누락): 42문제 — fallback 처리
  - `^` 소유격 표기: 2문제 — 아포스트로피로 변환
